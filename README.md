# A streamlined sort-to-sequence workflow for optimal single cell RNA sequencing
Repository containing the code necessary to generate figures for the Chip Sort manuscript.

# Downloading data
The data necessary to generate the figures can be downloaded from [our Zenodo record](https://zenodo.org/records/20345835).

To execute the R markdown files included in the 02_scripts directory, these data should be downloaded from Zenodo and unzipped into the 01_input directory.
